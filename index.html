Connect Google Calendar

You need a free Google Cloud Client ID to allow this app to read and write your calendar. It takes about 2 minutes to set up.

How to get your Client ID:
1. Go to console.cloud.google.com
2. Create a project → Enable Google Calendar API
3. Go to Credentials → Create → OAuth 2.0 Client ID
4. Type: Web application
5. Add your app URL to Authorised JavaScript origins
6. Copy the Client ID and paste it below
